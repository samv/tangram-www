Tangram from CVS

The latest development version of Tangram is available via CVS from the perlfect.com CVS repository.

To access, you will need to get a working CVS client, such as command line CVS, or perhaps you would prefer to use a CVS plug-in to your favourite graphical shell, such as Cervisia.

The CVSROOT is: :pserver:anonymous@perlfect.com:/home/cvsroot/

If you already have a recent GNU based operating system, you can likely just issue the following command:

cvs -d:pserver:anonymous@perlfect.com:/home/cvsroot checkout t2

There is no password for anonymous access - when prompted for a password, just press enter.
