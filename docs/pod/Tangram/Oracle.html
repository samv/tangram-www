NAME

Tangram::Oracle - Orthogonal Object Persistence in Oracle databases

SYNOPSIS

   use Tangram;
   use Tangram::Oracle;
   $schema = Tangram::Oracle->schema( $hashref );
   Tangram::Oracle->deploy($schema, $dbh);
      
   $storage = Tangram::Oracle->connect( $schema,
      $data_source, $username, $password );
   $storage->disconnect();
   Tangram::Oracle->retreat($schema, $dbh);

DESCRIPTION

This is the entry point in the Oracle-specific object-relational persistence backend.
